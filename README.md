# Tableau Sales & Customer Performance Dashboard

## Overview
This repository contains an interactive Tableau dashboard project completed as part of my Tableau learning journey.
I recreated this project by following the Tableau Ultimate Full Course by Data with Baraa on YouTube. While the dashboard concept and business requirements are based on the course, I independently built every worksheet, calculation, visualization, and dashboard component in Tableau.
The goal of this project was to strengthen my Tableau development skills and gain hands-on experience designing interactive business dashboards.

## Dashboard Objectives
The project consists of **two interactive dashboards**:
- Sales Performance Dashboard
- Customer Performance Dashboard
Together, they provide insights into sales performance, profitability, customer behavior, and business trends through interactive visual analytics.

## Sales Performance Dashboard
## Purpose
Analyze year-over-year sales performance through interactive KPI tracking and trend analysis.
## Highlights
- Year-over-year comparison of Sales, Profit, and Quantity
- Monthly sales trends with highest and lowest performance identified
- Product subcategory analysis with Sales vs. Profit comparison
- Weekly Sales and Profit performance with average reference lines

## Customer Performance Dashboard
## Purpose
Analyze customer behavior, purchasing patterns, and profitability through interactive visualizations.
## Highlights
- Year-over-year comparison of Customers, Sales per Customer, and Orders
- Monthly customer trends
- Customer distribution by number of orders
- Top 10 customers ranked by profit with key business metrics

## Dataset
Sales Dataset (provided as part of the learning project)
